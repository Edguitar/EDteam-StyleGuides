  • Curso
  • Para iconos que deban cambiar de color junto con el contenido, use elementos svg con la clase svg-icon.
  • Agregue la clase before o after segun la posición del código para añadir los márgenes correspondientes.
  • Use las clase cross-center cuando requiera forzar la alineación vertical con texto.
  • El icono tomará automáticamente el color del texto al que acompaña.
